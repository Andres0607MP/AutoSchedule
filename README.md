## *Andrey (disponibilidades, filtros avanzados y pruebas)*

Se trabaja en la parte que sirve de base para el algoritmo: la disponibilidad de los usuarios. Su trabajo asegura que cada persona pueda definir sus horarios libres, ya sea por días recurrentes o por fechas específicas. Construyendo los endpoints completos de Availability, los modelos, los serializers y las reglas de validación. También implementa el sistema de filtros avanzados con django-filter, permitiendo búsquedas como horarios, fechas, coincidencias por nombre y más. Su rol es clave para que el algoritmo reciba datos limpios y para que el usuario pueda ver su información organizada. Finalmente apoya de manera importante en la cobertura de pruebas del proyecto.
